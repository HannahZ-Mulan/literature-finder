# Literature Finder - 前端演示

## 🚀 服务器信息

**开发服务器运行在：** http://localhost:3005

## 📱 页面导航

### 1. 主页 (/)
- **未登录状态：**
  - 显示"Literature Finder"标题
  - 显示"登录"和"注册"按钮
  - 简洁的欢迎界面

- **已登录状态：**
  - 搜索框
  - 快捷入口卡片：
    - 我的库
    - 阅读列表
    - 分类与标签

### 2. 登录页面 (/login)
```
URL: http://localhost:3005/login
```
功能：
- 邮箱输入
- 密码输入
- 登录按钮
- 跳转到注册页面
- 登录成功后自动跳转主页

### 3. 注册页面 (/register)
```
URL: http://localhost:3005/register
```
功能：
- 姓名输入
- 邮箱输入
- 密码输入（最少6位）
- 确认密码
- 密码匹配验证
- 注册成功后自动跳转主页

### 4. 文献库页面 (/library)
```
URL: http://localhost:3005/library
```
功能：
- 搜索栏（标题、作者、摘要）
- 文献列表（卡片式）
- 分页导航
- 文献统计显示

## 🎨 UI特点

### 导航栏
- 左侧：Logo + 导航链接
- 右侧：用户信息 + 登出按钮（已登录）/ 登录注册按钮（未登录）

### 文献卡片
- 标题（可点击）
- 作者列表
- 摘要预览（3行）
- 元信息：
  - 期刊名称
  - 发表年份
  - 引用数
- 关键词标签
- 笔记显示（如果有）
- 操作按钮：
  - 查看详情
  - 生成摘要
  - 导出引用
