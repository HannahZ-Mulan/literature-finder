   - 检查结果是否显示

   - 搜索 "neural networks"

   - 点击"搜索历史"按钮
   - 点击一条历史记录
   - 检查是否自动搜索

   - 点击任意论文的"保存"按钮
   - 检查是否成功

## 📝 注意事项

1. **测试模式**: 当前使用测试模式，不需要真实登录
2. **API限制**: 某些学术数据库有速率限制，快速连续搜索可能遇到延迟
3. **网络要求**: 需要稳定的网络连接才能访问学术数据库API
4. **浏览器缓存**: 如果遇到问题，尝试刷新页面或清除缓存
5. **控制台日志**: 打开开发者工具查看详细的日志信息
