# Prompt更新记录

## 2026-04-02: 严肃但易读风格

### 更新内容

将AI解读风格从"人性化+小红书风格"改为"严肃但易读"，强调：
- 理性、克制、有洞察
- 拒绝网红和鸡汤
- 优先解释"为什么"而非"做了什么"

### 更新文件

1. **中文解读Prompt** (`src/lib/ai/chinese-summary-prompt.ts`)
   - 更新 `getHumanChineseSummaryPrompt` 函数
   - 新增写作风格要求：克制、理性、简洁
   - 强调解释"为什么"而非描述"做了什么"

2. **OpenAI对话Prompt** (`src/lib/ai/openai.ts`)
   - 更新 `chatWithPaper` 方法的 systemMessage
   - 5个核心要求：自然中文、不编造、解释原因、简洁有信息量、学术严谨

3. **DeepSeek对话Prompt** (`src/lib/ai/deepseek.ts`)
   - 同步更新对话系统提示词
   - 与OpenAI保持一致的风格要求

### Prompt核心要求

#### 中文解读
```
【写作风格】
- 克制、理性，不夸张
- 句子简洁，避免冗长
- 可以适当解释，但不要啰嗦
- 不使用网络流行语或营销语气

【重点】
请优先解释"为什么"，而不是仅描述"做了什么"。
```

#### 对话助手
```
1. 使用自然、清晰的中文
2. 不要编造信息
3. 优先解释原因和逻辑
4. 回答要简洁，但有信息量
5. 保持学术严谨
```

### 输出结构

1. 一句话总结
2. 研究在解决什么问题
3. 作者是怎么做的
4. 最重要的发现（2-3点）
5. 这篇论文的意义
6. 局限或需要注意的地方

### 禁止项

- 不要使用英文
- 不要使用markdown符号（* # - 等）
- 不要复制原文句子
- 不要编造信息
- 不要使用网络流行语

### 测试方法

访问 http://localhost:3000/paper/20
1. 点击"人性化解读"按钮
2. 测试AI对话功能
3. 检查输出是否符合新的风格要求
