- ✅ /api/recommendations - AI 智能推荐

### 前端 (Pages)
- ✅ / - 首页

### 第三方集成
- ✅ arXiv API
- ✅ PubMed API

- ✅ 手动格式化实现
- ✅ 5种格式支持：APA, MLA, Chicago, Harvard, Vancouver
- ✅ 从 DOI 自动获取页码
- ✅ Sentence case 转换
- ✅ Harvard 格式 "and" 连接词
- ✅ 批量导出功能

- ✅ GLM API 客户端 (支持真实 API 和 Mock 模式)

- ✅ 摘要缓存机制 (避免重复生成)
- ✅ 前端 UI 集成 (长度选择、生成按钮、展示区域)
